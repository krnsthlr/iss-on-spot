This website was built for the Full Stack JavaScript Program on Treehouse. It features "ISSOnSpot", a Twitter account that sends information about the next time the International Space Station (ISS) passes a specific location. Here is how it works:

  1. Log in to Twitter and send a tweet to @ISSOnSpot.
  2. Be sure to add a location to your tweet. If you need help with that see the Twitter Help Center .
  3. ISSOnSpot looks up the geographic coordinates of your location on Google Maps. If coordinates are found, it goes to Open Notify and gets the next ISS pass time for that location.
  4. Your local time is computed using the Google Timezone API. ISSOnSpot then notifies you about the next ISS fly-over time.
